# מערכת ניהול לקוחות - CRM

מערכת ניהול לקוחות ומשימות Full-Stack עם Angular 17, Angular Material, Node.js, Express, MongoDB ואימות JWT.

## תכונות

- ניהול לקוחות עם פעולות CRUD
- ניהול משימות והקצאות
- דשבורד רספונסיבי עם Angular Material
- תמיכה ב-PWA ועבודה אופליין
- Route Guards ו-HTTP Interceptors

## טכנולוגיות

| טכנולוגיה | שימוש |
|-----------|-------|
| Angular 17 | פריימוורק Frontend |
| Angular Material | ספריית UI |
| TypeScript | בטיחות טיפוסים |
| RxJS | תכנות ריאקטיבי |
| Node.js + Express | שרת Backend |
| MongoDB | בסיס נתונים |

## התקנה

```bash
git clone https://github.com/eladjak/Custome-Mengment-Angular.git
cd Custome-Mengment-Angular
npm install
ng serve
```
